Tech stack
----------
- Python 3.11+ (backend)
- FastAPI + uvicorn (stateless API)
- presidio-analyzer + presidio-anonymizer + spaCy (PII detection)
- React + Vite + TypeScript (frontend)
- pytest + pytest-cov + httpx (backend testing)
- Playwright MCP (frontend E2E testing)
- uv (Python dependency management)
- Supabase Auth (deferred — not in MVP)

Architecture
------------
- Stateless server: mapping returned to frontend, sent back on unmask
- Frontend: React SPA with Liquid Glass (iOS 26) styling
- Backend: FastAPI wrapping existing src/masker.py + src/validation.py
- History: localStorage (originalText + maskedText + mapping per entry)
- Deploy: Vercel (frontend) + Railway (backend)
